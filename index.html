JavaScript 核心邏輯基礎

1. 變數與資料型態

變數是存放資料的容器。let 可變動，const 不可變動。
常見型態：數字 (Number)、字串 (String)、布林 (Boolean)。

let score = 80;
score = 90;
console.log(score);

2. 流程控制 (判斷式與迴圈)

if...else 用來處理條件分支；for 迴圈用來處理重複的工作。

let i = 0;
if(i > 0) { console.log("A"); } else { console.log("B"); }

3. 函式 (Functions)

函式將程式碼打包，透過 ( ) 傳入參數，並由 return 回傳結果。

const multiply = (a) => { return a * 2; };
console.log(multiply(5));

4. 物件 (Object)

物件使用 { } 包裹，由「鍵 (Key)」與「值 (Value)」組成。

let car = { color: "Red" };

JS 進階概念

1. 函式的三種寫法

// 傳統寫法
function add(n1, n2){ return n1 + n2; }
// 變數指定
let add2 = function(n1, n2){ return n1 + n2; };
// 箭頭函式
const add3 = (n1, n2) => { return n1 + n2; };

2. JSON 與參數物件傳遞

function multiply(args){ return args.n1 * args.n2; }
let ans = multiply({ n1: 3, n2: 5 });
console.log(ans); // 15

3. 物件打包多個回傳值

function addAndMultiply(args){
    return { addAns: args.n1 + args.n2, mulAns: args.n1 * args.n2 };
}
let ans = addAndMultiply({ n1: 3, n2: 5 });
console.log(ans.mulAns);
